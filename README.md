# Note-Taker
**Note-Taker web application that manages a collection of Notes with the basic feature: list, insert, update, delete (or CRUD operations - Create, Update, Read and Delete).**

### Technologies Used :
- Servlet
- JSP
- MySQL(For DataBase)
- HiberNate(JPA)
